# 🏦 Action Cheeti System

A full-stack chit fund management system built using Spring Boot.

## 🚀 Features
- 👥 Add Members
- 💰 Monthly Payment Tracking
- 📊 Bidding System
- 🏆 Winner Selection
- 💸 Interest Distribution
- 📈 Dashboard (Real-time stats)

## 🛠 Tech Stack
- Java
- Spring Boot
- Hibernate (JPA)
- MySQL
- HTML, CSS, JavaScript

## 📊 Dashboard Includes
- Total Members
- Paid Members
- Total Bid Amount
- Winners Count

## ▶️ How to Run
1. Clone repository  
2. Open in Eclipse / IntelliJ  
3. Run `ActioncheetiApplication.java`  
4. Open browser:  
   http://localhost:8075
